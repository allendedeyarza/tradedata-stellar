# TradeData – Stellar (XLM)

Proyecto de Big Data centrado en el análisis de datos de trading de la criptomoneda
Stellar (XLM), utilizando TradingView y el ecosistema Hadoop.

## Criptomoneda principal
- Nombre: Stellar
- Símbolo: XLM
- Exchange: BINANCE
- Par de trading: XLMUSDT

## Objetivo
Diseñar e implementar una arquitectura Big Data que permita:
- Ingesta de datos históricos (batch)
- Procesamiento y análisis con Spark
- Preparación para datos en tiempo real
